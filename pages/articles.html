مقدمه‌ای بر هوک‌های React

هوک‌ها در React 16.8 معرفی شدند و انقلابی در نحوه نوشتن کامپوننت‌های تابعی ایجاد کردند. با استفاده از هوک‌ها می‌توانید state و سایر ویژگی‌های React را بدون نوشتن کلاس استفاده کنید.

نکته مهم
هوک‌ها backward-compatible هستند و جایگزین دانش شما از مفاهیم React نمی‌شوند. بلکه راه‌جدیدی برای استفاده از ویژگی‌های آشنا ارائه می‌دهند.

هوک‌ها مشکلات زیر را حل می‌کنند:

  • پیچیدگی‌های ناشی از استفاده از کلاس‌ها
  • مشکل wrapper hell در استفاده از HOCها
  • سردرگمی در کار با this
  • جداسازی منطق مرتبط در کامپوننت‌ها

هوک useState

هوک useState به شما امکان می‌دهد state را در کامپوننت‌های تابعی اضافه کنید. این هوک یک جفت مقدار برمی‌گرداند: مقدار فعلی state و تابعی برای به‌روزرسانی آن.

JavaScript 
import { useState } from 'react';

function Counter() {
  // مقدار اولیه 0 و تابع به‌روزرسانی
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>شما {count} بار کلیک کرده‌اید</p>
      <button onClick={() => setCount(count + 1)}>
        افزایش شمارنده
      </button>
    </div>
  );
}

مثال پیشرفته با useState

می‌توانید از useState برای مدیریت stateهای پیچیده‌تر نیز استفاده کنید:

JavaScript 
function Form() {
  const [formData, setFormData] = useState({
    username: '',
    email: '',
    password: ''
  });

  const handleChange = (e) => {
    const { name, value } = e.target;
    setFormData(prev => ({
      ...prev,
      [name]: value
    }));
  };

  return (
    <form>
      <input
        name="username"
        value={formData.username}
        onChange={handleChange}
      />
      {/* سایر فیلدها */}
    </form>
  );
}

جمع‌بندی

در این مقاله به بررسی جامع هوک‌های اصلی React پرداختیم. هوک‌ها ابزارهای قدرتمندی هستند که کدنویسی با React را ساده‌تر و خوانا‌تر می‌کنند.

نکات کلیدی:

  • از useState برای مدیریت state استفاده کنید
  • از useEffect برای اثرات جانبی استفاده کنید
  • هوک‌های سفارشی برای استفاده مجدد از منطق ایجاد کنید
  • قوانین هوک‌ها را رعایت کنید
