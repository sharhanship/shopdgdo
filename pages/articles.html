آموزش هوک‌های React

نویسنده: علی محمدی تیر ۱۴۰۲

چکیده

هوک‌ها در React 16.8 معرفی شدند و امکان استفاده از state و ویژگی‌های دیگر را در کامپوننت‌های تابعی فراهم کردند. در این مقاله به بررسی useState، useEffect و useContext می‌پردازیم.

useState

برای مدیریت state در کامپوننت‌های تابعی استفاده می‌شود.

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <button onClick={() => setCount(count + 1)}>
        تعداد: {count}
      </button>
    </div>
  );
}

useEffect

برای مدیریت اثرات جانبی در کامپوننت‌ها استفاده می‌شود.

import { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `شما ${count} بار کلیک کردید`;
  }, [count]);

  return <button onClick={() => setCount(count + 1)}>
    کلیک کنید
  </button>;
}
