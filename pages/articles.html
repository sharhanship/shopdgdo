ساخت یک سیستم ورود پیشرفته با طراحی خاص

منتشر شده توسط تیم Hyperion

در این آموزش قصد داریم یک فرم ورود پیشرفته طراحی کنیم. ابتدا ساختار HTML را می‌سازیم:

<form action="#" method="post">
  <input type="text" placeholder="نام کاربری">
  <input type="password" placeholder="رمز عبور">
  <button type="submit">ورود</button>
</form>

<form action="#" method="post">
  <input type="text" placeholder="نام کاربری">
  <input type="password" placeholder="رمز عبور">
  <button type="submit">ورود</button>
</form>

حالا با استفاده از CSS، فرم را استایل می‌دهیم:

form {
  background: rgba(255,255,255,0.1);
  backdrop-filter: blur(10px);
  border-radius: 15px;
  padding: 20px;
}
